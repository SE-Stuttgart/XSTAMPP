Refined Safety View

This View will show any entry which is marked as "Hazardous" in any context.
You can link any unsafe control actions with these entries, by clicking into the cell "Related Unsafe CA".
This will open a small Dialog with all available "Unsafe Control Actions". Then you can link them, similar to the dependencies view. As soon as this table has been opened xstpa will automatically generate refined rules out of the hazardous combinations these can be seen in the "generated Rules" column You will find three Buttons on the side of the Table. The first one creates the "LTL Table", which is currently only used for export, but it will be enhanced in the future. The second Button opens the Export Dialog which is the same as in XSTAMPP.
