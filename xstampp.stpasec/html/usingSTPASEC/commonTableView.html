Working with Losses, Vulnerabilities, Security Constraints, Design Requirements and Control Actions

Adding and Deleting

Select one of the mentioned views in the navigation bar. The navigation bar is the left section of the STPA for Security window, it splits the views into the three steps Analysis Fundamentals, Unsecure Control Actions and Causal Analysis. After selecting the desired view, click on to add an item to the list. can be used to delete an item or multiple items by selecting more than one item by holding the shift button. You will be prompted to confirm the deletion of the selected items.

Editing the Title and Description

The title of a selected item can be edited by doubleTclicking on the existing item title or pressing enter. The item description of the selected item is shown in the right section of the STPA for Security window. It can be edited by simply clicking on the description text.
